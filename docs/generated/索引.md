# 生成文档索引

`docs/generated/` 用于承接由脚本、模型或源码派生出的辅助文档。

## 入口

- [docs 总索引](../索引.md)

## 使用规则

- 生成文档应尽量可重复产出
- 生成文档不是源码真相源，源码或设计文档仍应保持主导地位
- 若生成结果长期需要人工修订，应考虑拆分为“生成源 + 解释文档”
