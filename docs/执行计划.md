## 执行计划规则

- 当前仓库默认只维护一份 active 计划。
- 只有当新的业务阶段真正启动，且当前计划已不足以承载时，才拆出下一份 active 计划。
- 已完成阶段统一收敛到归档总览或新的专项归档，不恢复到大量并列清单结构。
