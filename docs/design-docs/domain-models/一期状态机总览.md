# 数字藏品运营与展示平台一期状态机总览

## 文档目标

只保留状态名称、允许流转和跨状态协同约束。

## 真相源

- 枚举与模型：`packages/shared-types/src/**`、`apps/api/prisma/schema.prisma`
- 状态实现：`apps/api/src/modules/**`

## 总规则

- 资产状态、内容状态、审核状态分开存
- 状态切换只能由明确业务动作触发
- 审核历史必须可追溯
- 公开可见性由发布状态控制

## 激活码状态

- `UNISSUED`
- `ISSUED`
- `USED`
- `VOIDED`
- `EXPIRED`

允许流转：

- `UNISSUED -> ISSUED | VOIDED | EXPIRED`
- `ISSUED -> USED | VOIDED | EXPIRED`

## 藏品状态

- `PENDING_CLAIM`
- `OWNED`
- `FROZEN`

允许流转：

- `PENDING_CLAIM -> OWNED`
- `OWNED <-> FROZEN`

## 内容编辑状态

- `DRAFT`
- `UNDER_REVIEW`
- `REJECTED`
- `APPROVED`

## 内容发布状态

- `UNPUBLISHED`
- `PUBLISHED`
- `TAKEDOWN`

允许流转：

- `UNPUBLISHED -> PUBLISHED`
- `PUBLISHED -> TAKEDOWN`
- `TAKEDOWN -> PUBLISHED`

## 内容审核状态

- `PENDING_MACHINE`
- `MACHINE_APPROVED`
- `MACHINE_REJECTED`
- `PENDING_MANUAL`
- `MANUAL_APPROVED`
- `MANUAL_REJECTED`

## 评论状态

- `PENDING_MACHINE`
- `MACHINE_APPROVED`
- `MACHINE_REJECTED`
- `PENDING_MANUAL`
- `MANUAL_APPROVED`
- `MANUAL_REJECTED`
- `BLOCKED`

## 转让单状态

- `PENDING_ACCEPT`
- `COMPLETED`
- `CANCELLED`
- `EXPIRED`

允许流转：

- `PENDING_ACCEPT -> COMPLETED | CANCELLED | EXPIRED`

## 会员状态

- `ACTIVE`
- `FROZEN`

## 协同约束

- 激活码 `USED` 时，对应藏品必须已 `OWNED`
- 会员被冻结时，不自动改变已公开内容状态
- 内容审核状态和发布状态必须分别存储
- 新版本审核中时，旧公开版本可继续在线
- 转让完成后拥有者变化，但公开页地址保持不变
