# 数字藏品运营与展示平台接口契约草案

## 文档目标

给 AI 和研发一个“先看什么”的短入口，不再在本文重复展开所有 DTO 细节。

## 真相源

- 字段级真相源：`packages/api-contracts/src/**`
- HTTP 实现真相源：`apps/api/src/modules/**`
- 本文用途：说明接口边界、核心资源和阅读顺序

## 统一约定

- 后台接口前缀：`/admin-api`
- 会员接口前缀：`/member-api`
- 公开接口前缀：`/public-api`
- 会员接口默认使用 `Authorization: Bearer <memberAccessToken>`
- 成功响应统一走 `success + data`
- 失败响应统一走 `success + error`
- 分页列表统一返回 `items + page + pageSize + total`

示例：

```json
{
  "success": true,
  "data": {}
}
```

```json
{
  "success": false,
  "error": {
    "code": "RESOURCE_NOT_FOUND",
    "message": "..."
  }
}
```

## 先读哪些接口

### `admin-api`

- `auth`：后台登录与当前身份
- `series`：系列管理
- `issuance-batches`：发行批次管理
- `activation-codes`：激活码生成与查询
- `collections`：藏品列表与详情
- `collection-reviews`：内容审核、审核历史、人工通过/驳回/下架
- `members`：会员查询、冻结、解冻
- `dashboard` / `notifications` / `transfers`：M4 运营侧能力

### `member-api`

- `auth`：微信登录、当前会员信息
- `collection-activation`：激活并领取藏品
- `my/collections`：我的藏品、内容草稿、提交审核
- `my/messages`：会员消息中心
- `my/transfers`：会员转让记录、发起转让、接收转让
- `collection-comments`：发表评论与回复

### `public-api`

- `collections`：公开展示页
- `collections/:slug/comments`：公开评论列表

## 主链路最小接口集

### M1 最小发行闭环

- `POST /admin-api/auth/login`
- `GET /admin-api/series`
- `POST /admin-api/series`
- `GET /admin-api/issuance-batches`
- `GET /admin-api/issuance-batches/:batchId`
- `POST /admin-api/issuance-batches`
- `GET /admin-api/activation-codes`
- `POST /admin-api/activation-codes/generate`
- `GET /admin-api/collections`
- `POST /member-api/auth/wechat-miniapp`
- `GET /member-api/auth/me`
- `POST /member-api/collection-activation`
- `GET /member-api/my/collections`

### M2 内容展示闭环

- `GET /member-api/my/collections/:collectionId/content`
- `POST /member-api/my/collections/:collectionId/content/drafts`
- `POST /member-api/my/collections/:collectionId/content/submissions`
- `GET /public-api/collections/:slug`

### M3 审核治理闭环

- `GET /admin-api/collection-reviews`
- `POST /admin-api/collection-reviews/:reviewId/approve`
- `POST /admin-api/collection-reviews/:reviewId/reject`
- `GET /admin-api/collection-reviews/history`
- `POST /admin-api/content-versions/:contentVersionId/takedown`
- `GET /admin-api/members`
- `PATCH /admin-api/members/:memberId/status`

### M4 通知与流转闭环

- `GET /admin-api/dashboard/overview`
- `GET /admin-api/notifications/overview`
- `GET /admin-api/transfers`
- `GET /member-api/my/messages`
- `GET /member-api/my/transfers`
- `POST /member-api/my/collections/:collectionId/transfers`
- `POST /member-api/my/transfers/:transferId/accept`

## 常见错误码分组

- 鉴权类：`UNAUTHORIZED`、`FORBIDDEN`、`MEMBER_ACCOUNT_FROZEN`
- 参数类：`VALIDATION_ERROR`
- 资源类：`RESOURCE_NOT_FOUND`、`SERIES_NOT_FOUND`、`BATCH_NOT_FOUND`
- 状态类：`INVALID_STATUS_TRANSITION`、`REVIEW_ALREADY_FINALIZED`
- 业务类：`ACTIVATION_CODE_ALREADY_USED`、`COLLECTION_NOT_OWNED`
- 展示类：`PUBLIC_COLLECTION_TAKEDOWN`
