# 平台 PRD

## 文档信息

- 文档类型：产品需求文档（PRD）
- 当前阶段：需求讨论稿
- 文档目标：明确一期产品范围、核心流程、业务规则、模块能力与关键约束，为后续设计、研发、测试提供统一基线。

## 产品背景与目标

### 产品背景

平台需要建设一套围绕数字藏品的运营与展示系统，支持后台发行数字藏品、通过激活码发放给 C 端会员、由会员对藏品展示内容进行编辑和公开展示，并支持评论互动、内容审核、资产转让及消息通知。

### 产品目标

- 建立完整的数字藏品发行与领取链路。
- 建立 C 端会员藏品内容管理与展示能力。
- 建立内容审核、评论审核、人工复核与风控治理能力。
- 建立藏品转让能力。
- 建立统一通知中心。
- 建立后台权限与运营管理能力。

### 一期范围目标

- 实现“发码 -> 领藏品 -> 编辑内容 -> 审核 -> 展示 -> 评论 -> 转让 -> 通知”闭环。
- 优先完成核心业务链路，不在一期内做复杂交易、复杂内容编辑器和深度数据分析。

本节保留为总览入口，角色定义专题见：

- [用户角色与权限视角](./user-roles-and-personas.md)

摘要：

- 后台角色覆盖超级管理员、发行运营、内容审核员、评论审核员、会员运营/客服

## 业务对象定义

### 系列

用于定义一类藏品的上层分类，是发行批次的归属对象。

### 发行批次

用于定义某个系列下的一次实际发行行为，承载发放数量、有效期、批次状态等信息。

### 激活码

用户领取藏品的唯一凭证。生成激活码时，系统同步生成对应藏品。

### 藏品

数字资产本体。激活前处于待领取状态，激活后归属到会员名下。

### 藏品内容版本

藏品的展示内容实体。用于承载用户编辑后的内容，并支持审核、公开、下架、版本替换。

### 会员

C 端用户账户。支持微信小程序与公众号授权登录，统一归并为同一会员账号。

### 评论

会员对公开藏品展示内容发表的互动内容，支持二级评论，需经过审核。

### 转让单

一笔藏品转让流程实例，用于承载转让方式、转出方、转入方、状态、有效期等信息。

### 通知消息

系统向会员发送的业务消息，支持站内信与站外通知两种形式。

## 业务流程设计

本节保留为主链路总览，详细流程专题见：

## 后台功能需求

### 仪表盘

目标：为运营、审核、客服提供关键业务总览与待办提醒。

功能要求：

- 展示激活码总量、已使用数、待发放数。
- 展示藏品总量、已领取数、冻结数。
- 展示待人工复核内容数。
- 展示待审核评论数。
- 展示待接收转让数。
- 提供近期激活、评论、审核趋势。
- 提供待办列表：
  - 待人工复核内容
  - 待审核评论
  - 待接收转让
  - 即将失效激活码或批次

### 系列管理

功能要求：

- 系列列表查询。
- 新增系列。
- 编辑系列。
- 启用或停用系列。

核心字段：

- 系列名称
- 系列标识
- 系列描述
- 状态
- 创建时间

### 发行批次管理

功能要求：

- 批次列表查询。
- 新增批次。
- 编辑批次。
- 启用或停用批次。
- 按批次生成激活码。

核心字段：

- 批次名称
- 批次编号
- 所属系列
- 发行数量
- 激活码有效期
- 状态
- 备注

业务规则：

- 批次停用后不可继续生成激活码。
- 已生成未使用激活码仍可继续激活。

### 激活码管理

功能要求：

- 激活码列表。
- 批量生成。
- 导出激活码。
- 批量标记发放。
- 作废未使用激活码。
- 查看激活或发放记录。

核心字段：

- 激活码
- 所属系列
- 所属批次
- 对应藏品编号
- 状态
- 发放渠道
- 使用会员
- 使用时间
- 失效时间

### 藏品管理

功能要求：

- 藏品列表查询。
- 按系列、批次、状态、会员筛选。
- 查看藏品详情。
- 冻结或解冻藏品。
- 查看当前公开内容与待审核版本。
- 查看评论统计。
- 查看审核记录。
- 查看转让记录。

核心字段：

- 藏品编号
- 系列
- 发行批次
- 所属会员
- 藏品状态
- 内容发布状态
- 内容审核状态
- 激活时间

### 内容列表与人工复核

功能要求：

- 查看所有藏品内容状态。
- 按系列、批次、会员、状态筛选。
- 查看当前公开版本。
- 查看待审版本。
- 人工复核疑似异常内容。
- 人工通过或驳回并下架。
- 填写驳回原因。
- 展示完整审核历史。

审核记录需展示：

- 提交时间
- 机审结果
- 命中规则
- 人工审核人
- 审核结论
- 驳回或下架原因
- 处理时间

### 评论管理与评论审核

功能要求：

- 评论列表查询。
- 按藏品、会员、状态筛选。
- 查看评论上下文。
- 屏蔽或删除评论。
- 审核待人工审核评论。
- 审核通过或驳回。

核心字段：

- 评论内容摘要
- 藏品编号
- 所属会员
- 父评论
- 状态
- 提交时间

### 会员管理

功能要求：

- 会员列表查询。
- 查看会员详情。
- 冻结或解冻会员。
- 查看名下藏品。
- 查看名下内容。
- 查看评论记录。
- 查看绑定渠道信息。

会员详情展示：

- 基础信息
- 渠道绑定信息
- 名下藏品
- 内容状态
- 评论行为
- 转让记录
- 风险处理记录

### 转让记录管理

功能要求：

- 转让记录列表。
- 按藏品、转出会员、转入会员、方式、状态筛选。
- 查看转让详情。
- 必要时后台取消未完成转让单。

核心字段：

- 转让单 ID
- 藏品编号
- 转让方式
- 转出会员
- 转入会员
- 转让码
- 有效期
- 状态
- 发起时间
- 完成时间

### 通知中心

功能要求：

- 消息模板管理。
- 站内消息记录查询。
- 站外通知记录查询。
- 查看发送状态与失败原因。

通知渠道：

- 站内信
- 微信小程序订阅消息
- 公众号通知

一期核心事件：

- 激活成功
- 内容审核通过
- 内容审核失败
- 内容被人工下架
- 评论审核失败
- 转让发起
- 收到转让
- 转让成功
- 转让失效

### 系统管理

包含：

- 后台用户管理
- 角色管理
- 菜单管理
- 字典管理
- 登录日志
- 操作日志

## C 端功能需求

### 登录与账户体系

- 支持微信小程序授权登录。
- 支持微信公众号授权登录。
- 统一会员账户。
- 优先按 unionid 归并。
- 无法直接归并时支持手机号补充归并。

### 激活我的藏品

功能要求：

- 输入激活码。
- 校验激活码有效性。
- 激活成功反馈。
- 跳转我的藏品。

### 我的藏品

功能要求：

- 查看本人名下藏品列表。
- 查看藏品状态。
- 进入内容编辑。
- 提交审核。
- 清空内容。
- 发起转让。
- 查看转让记录。

### 藏品内容编辑

功能要求：

- 编辑展示标题、封面、简介。
- 内容字段与配置字段先预留。
- 设置是否允许评论。
- 保存草稿。
- 提交审核。
- 查看驳回原因。
- 清空内容。

业务规则：

- 清空内容不删除藏品实体。
- 清空操作仅作用于编辑版本。
- 已公开旧版本在新版本未生效前继续在线。

### 藏品公开展示页

功能要求：

- 公开访问。
- 展示藏品内容。
- 展示藏品编号、系列、发行批次等基础信息。
- 展示浏览量与评论数。
- 点赞位预留。
- 登录用户可评论。
- 所有者可进入编辑入口。

异常规则：

- 内容不存在、未公开、已下架时，展示统一兜底页。

### 评论区

功能要求：

- 支持一级评论与二级回复。
- 未登录用户不可评论。
- 支持评论审核反馈。

### 转让功能

功能要求：

- 发起指定会员转让。
- 发起转让码转让。
- 查看转让状态。
- 接收指定转让。
- 输入转让码接收转让。

业务规则：

- 有草稿或待审核内容的藏品不可转让。
- 同一时间只能存在一个有效转让单。
- 转让成功后归属与编辑权变更。

### 消息中心

功能要求：

- 消息列表。
- 未读或已读。
- 类型筛选。
- 点击跳转业务详情。
- 标记已读。
- 不支持用户删除消息。

## 关键业务规则汇总

本节详细规则已整理到专题文档：

- [状态与业务规则汇总](./state-and-business-rules.md)

摘要：

- 后台与会员体系严格隔离

## 一期范围

- 完整覆盖发行、激活、藏品管理、内容审核、评论审核、转让、通知、权限、会员管理。

- 通知编排增强
  - 多模板策略
  - 分渠道重试
  - 用户触达偏好控制
- 风控增强

  - 黑名单与风险标签
