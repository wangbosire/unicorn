# 一期里程碑总览

本文档从执行计划中抽出里程碑视图，便于快速查看阶段目标、范围与验收口径。

## 当前状态

- 状态：`active`
- 上位计划：[V1 执行计划](./v1-exec-plan.md)

## M1：最小发行闭环

范围：

- 会员登录
- 后台权限底座
- 系列/批次
- 激活码生成
- 藏品生成
- 藏品激活领取

验收标准：

- 后台可生成批次激活码。
- 会员可成功激活并领取藏品。

补充文档：

- [M1 验收清单](./m1-acceptance-checklist.md)

## M2：内容展示闭环

范围：

- 藏品内容版本
- 内容编辑
- 提交审核
- 机审
- 公开展示页

验收标准：

- 会员可提交藏品内容并完成公开展示。

## M3：审核治理闭环

范围：

- 人工复核
- 审核历史
- 下架兜底页
- 会员管理
- 评论审核

验收标准：

- 平台可处理疑似异常内容和评论风险。

## M4：流转与通知闭环

范围：

- 转让体系
- 通知中心
- 仪表盘

验收标准：

- 藏品可完成转让。
- 关键事件具备站内外消息触达。
