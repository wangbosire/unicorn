# Miniapp

TaroJS 小程序应用骨架目录。

## 目标定位

会员端小程序承担一期 C 端主任务流，重点覆盖：

- 授权登录
- 激活我的藏品
- 我的藏品
- 藏品编辑
- 公开展示浏览
- 消息中心
- 转让

## 推荐页面分组

- `pages/auth`：登录、授权、绑定
- `pages/activate`：激活码输入与激活结果
- `pages/collections`：我的藏品、详情、编辑
- `pages/public`：公开展示页
- `pages/messages`：站内消息
- `pages/comments`：评论互动
- `pages/profile`：个人中心
